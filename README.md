# Medical Appointment No-Show Analysis

This project analyzes a medical appointment dataset to understand why some patients miss their scheduled appointments.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Research Questions
- What proportion of appointments result in no-shows?
- Does age affect attendance?
- Does receiving an SMS reminder affect attendance?
- Does waiting time influence no-show rates?

## Project Files
- medical_no_show_analysis.ipynb → Full analysis in Jupyter Notebook
- medical_no_show_analysis.html → Exported analysis report

## Project Goal
Identify patterns that influence patient attendance and provide insights that may help healthcare providers reduce missed appointments.
